Suspicious Links

What could happen if you open a fake website?

  • Malware could be injected into your system
  • Your personal information could be stolen
  • Phishing attacks
  • Identity theft
  • Financial loss

Common signs of Suspicious links

  • Unusual or misspelled domain names (Eg: gogle.com)
  • Links that do not match the displayed text
  • Links shared by unknown or untrusted sources
  • Links with unusual file extensions (Eg: .exe, .scr)
  • Shortened Links (Eg: bit.ly/2kT3sY)
  • Links that ask for personal information
  • Strange characters or symbols (Eg: faceb00k.com)
  • Extra words in the address (Eg: school-login.safe-access-verify.com)

How to check for a safe link

  1. Hover over the link before clicking it to confirm it leads you to the site you are expecting
  2. Check the URL protocol for HTTPS (Eg: https://www.example.com); never open websites starting with HTTP
  3. Use link scanning tools (Eg: VirusTotal, URLVoid) to check the safety of the link
  4. Search the website name on Google to confirm legitimacy
  5. Ask a trusting adult if you are still unsure
